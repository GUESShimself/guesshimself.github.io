# guesshimself.work

Personal portfolio site for Eric Guess — a design systems and product design professional.

**Live site:** [guesshimself.work](http://guesshimself.work/)

## About

Single-page portfolio featuring selected case studies, design principles, and capabilities. Built to showcase work for Design Systems Lead roles.

## Tech Stack

- Vanilla HTML, CSS, JavaScript — zero dependencies, no build tools
- Variable fonts: Inter (body) and Fraunces (headings)
- Mobile-first responsive design with three breakpoints
- Light/dark mode with system preference detection
- Experimental Le Corbusier-inspired "Modulor" skin (golden ratio proportional system)
- Deployed via GitHub Pages

## Structure

```
index.html          Single-page portfolio
css/                Modular stylesheets (reset, tokens, base, layout, components, responsive)
css/skins/          Experimental theme skins
js/                 Theme toggle, interactions, smooth scroll
```
